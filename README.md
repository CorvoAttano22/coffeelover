## Description

Coffee Shop API — NestJS Backend.

Backend API for a coffee e-commerce website, built with NestJS + TypeScript.
Created as a learning project to practice backend architecture and clean API design.

## Tech Stack

- Node.js / NestJS
- TypeScript
- PostgreSQL
- TypeORM
- REST API consumed by a Bootstrap 4 frontend (Colorlib template)
- API documentation available via Swagger at `/api/docs`

## Project setup

npm install

## Compile and run the project

# development

npm run start

# watch mode

npm run start:dev

# production

npm run start:prod
